About the programme

The Square root or (Odmocnina) program is a simple tool for calculating powers and square roots.
It does not yet have a graphical interface like a traditional calculator,
so it is controlled using the command line.
Requires Java 8,
which you can download if you don't have it installed on this page:
https://www.java.com/en/
To control the program there is a file Odmocnina.bat in each archive of the program.
The file starts the program and guides you through several operations
